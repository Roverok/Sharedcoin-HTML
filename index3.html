  • How does it work?

    Coins send with shared send will be matched up with another user. When a match is found your coins will be swapped breaking the transaction chain from your own wallet. Coins will be swapped with multiple users making the chain even harder to follow.

  • What logs are kept?

    Each shared transaction creates an entry in a forwarding or routing table which is used to track the final destination address and the value of the transaction. After 6 confirmations (~1 hour) the entry is permanently removed from database leaving no connection between the input and output address. Unused forwarding entries are removed after 8 hours.

  • How long do shared transactions take?

    Amounts under 10 BTC will be forwarded within 10 seconds providing they meet certain requirements (Relayed By greater > 90% of the network and no conflicting double spends)

    Amounts under 25 BTC require 1 confirmation (~10 minutes)

    Amounts under 250 BTC require 2 confirmations (~20 minutes)

    The above times are dependant on untainted coins being available in the system, some delay maybe incurred if fresh coins are needed, particularly for large amounts.

  • Is there a maximum transaction size?

    The maximum transaction size is 250 BTC. Transactions larger than the maximum will be returned to the sender.

  • Is there a minimum transaction size?

    We recommended you do not send transactions less than 0.2 BTC

  • What if there isn't sufficient coins available to process my transactions?

    The funds will wait in the mixer until new coins are available (Sent in randomized chunks). There is no maximum amount of time the funds can wait for but they will always come out the other side eventually.

  • How can you guarantee that the transaction chain will be broken?

    There is no guess work involved, each shared transaction analyzes up to 50,000 outputs or 250 levels deep in the blockchain to ensure the coins sent to the destination address are 100% untainted with the original coins.

    The source code for taint analysis calculations can be found on the Blockchain Github Project

  • Is there a way to verify the coins were properly mixed?

    You can use the taint analysis tool on blockchain.info to verify there is no connection between the input and output addresses. Enter the destination address in the search box on the blockchain.info homepage, on the address page click "View Taint Analysis". If the mixing service has worked properly you should not be able to find the input address on that page.

  • I haven't received my funds or have only received a partial amount!

    Please be patient all coins reach the other side eventually. If you would like us to send the coins back early please Contact Us with your deposit address.

  • I'm worried my transaction is taking a long time to confirm and may be removed before being forwarded

    Any mixer entries with pending transactions will never be removed, once confirm the transaction will be processed fine (even if it took longer than 8 hours to confirm).

  • Are there any fees for shared transactions?

    The fee for shared transactions is currently 0.5% fee.
